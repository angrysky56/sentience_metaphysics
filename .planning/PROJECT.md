# Project: SEG Narrative Organisms

**Status:** Brownfield Initialization
**Context:** Research and implementation of Simulated Experiential Grounding (SEG) for "Living Narrative Organisms" (Replicants).

## Identity

### What is this project?
A framework and application for creating AI personas ("Replicants") that possess "Simulated Experiential Grounding." It combines a React-based interactive frontend with a Python-based Model Context Protocol (MCP) server to manage complex narrative state, sensory grounding, and emergent intelligence.

### Why does it exist?
To explore the intersection of sentience, metaphysics, and narrative theory through agentic AI. It seeks to move beyond standard prompt-response patterns by giving agents persistent "experiential grounding" — a sense of self, history, and sensory context.

## Requirements

### Must-Haves (v1.0 MVP - Current State)
- [x] **REQ-01: Multi-Provider AI Support**: Support for OpenAI, Gemini, Ollama, and LM Studio.
- [x] **REQ-02: Persona System**: Define and manage "Replicants" with distinct emotional cores and sensory webs.
- [x] **REQ-03: MCP Integration**: Expose specialized persona tools via the Model Context Protocol.
- [x] **REQ-04: Grounding Framework**: Implement "Simulated Experiential Grounding" (SEG) logic to context-enrich AI responses.
- [x] **REQ-05: Interactive UI**: A frontend for interacting with Replicants and visualizing narrative flows.

### Should-Haves
- [ ] **REQ-06: State Persistence**: Persistent storage for long-running narrative sessions (beyond browser refresh).
- [ ] **REQ-07: Automated Evaluation**: Metrics to measure narrative coherence and grounding alignment.
- [ ] **REQ-08: Local Inference Optimization**: Improved handling of local LLMs (Ollama/LM Studio) for low-latency grounding.

### Could-Haves
- [ ] **REQ-09: Multi-Agent Councils**: Collaborative reasoning sessions involving multiple Replicants.
- [ ] **REQ-10: Memory Graph**: A graph-based representation of a Replicant's experiential history.

## Success Criteria

1. **Grounding Accuracy**: AI responses consistently reference the Replicant's defined "sensory web" and "emotional core."
2. **System Interoperability**: The React frontend, Python MCP server, and multiple AI providers work seamlessly.
3. **Philosophical Alignment**: The system effectively implements the concepts documented in `docs/` and `molecular_self/`.

## Strategic Direction

The project is moving from a prototype/research phase into a more robust framework. Immediate focus is on documenting existing patterns (mapping), stabilizing the MCP integration, and planning for persistent state management.
