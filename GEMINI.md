# GEMINI

<!-- GSD:project-start source:PROJECT.md -->
## Project

**SEG Narrative Organisms**
A framework for "Living Narrative Organisms" (Replicants) using Simulated Experiential Grounding. Combines React frontend with a Python MCP server to explore sentience and narrative theory through agentic AI.
<!-- GSD:project-end -->

<!-- GSD:stack-start source:STACK.md -->
## Technology Stack

- **Languages**: TypeScript 5.2 (Frontend), Python 3.10+ (Backend/MCP)
- **Frameworks**: React 18, Vite 5, MCP (Model Context Protocol)
- **UI/Styling**: Tailwind CSS, Radix UI, Framer Motion
- **Services**: AI Service (OpenAI, Gemini, Ollama, LM Studio)
<!-- GSD:stack-end -->

<!-- GSD:conventions-start source:CONVENTIONS.md -->
## Conventions

- **React**: Functional components, strict TypeScript, hooks-based logic.
- **Python**: PEP 8, type hints, async/await for I/O and MCP tools.
- **Patterns**: Service-oriented frontend, tool-based MCP backend.
- **Quality**: Trunk and ESLint for code consistency.
<!-- GSD:conventions-end -->

<!-- GSD:architecture-start source:ARCHITECTURE.md -->
## Architecture

- **Frontend**: SPA communicating with AI providers and local MCP server.
- **Backend**: Python MCP server providing persona logic and grounding tools.
- **Grounding**: "Molecular Self" framework for persona-driven context enrichment.
<!-- GSD:architecture-end -->
